Tool for Converting JSON and Text to RDF triples
        JSON2RDF
JSON2RDF method receives data as JSON format and converts to rdf representation
as N-Triples serialization.  


        Text2RDF
It receives textual data as input, and then tokenizes it to extract the keywords and
phrases from the text file to represent it as rdf triples.For text mining “stanford-corenlp4.1.0-” 
is used to process the textand prepare it for rdf based representation.
